How to Run the Program Without the Source Code
To run the program without accessing the coding files, please follow these steps:

1. Download the program archive (.rar file).
2. Extract the contents of the archive to your Desktop.
3. Open the extracted folder. You will see two items:

- A document file
- A program folder

4.Open the program folder. Inside, you will find the following:

- .vs folder
- FitnessClubManagementApp folder
- Program folder
- FitnessClubManagementApp.sln file

5. Open the Program folder. There you will find the program application file (the executable).
